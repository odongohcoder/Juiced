alerts

Alerts make it easy to display prominate information to the user. They are styled using colors from your palette. They can have exit buttons, but Juiced does not have any Javascript built in, so you must manually add any exit functionality. See how.

Add this JavaScript before your closing </body> tag for simple exit functionality: 
        <script>
        var _exits = [].slice.call(document.querySelectorAll( '.alert-exit' ));
        _exits.forEach(function(ele) {
             ele.addEventListener("click", function() {
                 var _parent = this.parentNode;
                  _parent.className = _parent.className + " display-none";
             });
        });
        </script>
What does this JavaScript do? It looks for all elements with a class of .alert-exit. When clicked, it finds the parent of the exit button, (so don't wrap your exits), and adds a class of .display-none to that parent, hiding the alert with the CSS display:none property.
